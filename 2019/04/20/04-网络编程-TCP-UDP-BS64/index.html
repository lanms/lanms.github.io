1. 1 TCP 可靠传输协议

  • 传输控制协议 Transfer Control Protocol

    • 可靠通信(数据不会穿丢也不会传错)
    • 流量控制 自动调节发送数据的速度
    • 拥塞控制 网络拥堵时会降低发送速率

  • 可靠通讯的实现:

    • 如何保证数据不会传输错误
    • 握手机制 + 在数据中添加冗余校验对发送的数据 进行校验

  • 流量控制:

    • 滑动窗口机制 逐步增加发送的数据大小

  • 拥塞控制:

    • 减小滑动窗口 减小发送速率

  • User Datagram Protocol 用户数据宝协议
