sys

  • 在Linux系统命令行参数 在输入命令是给的参数
  • sys.argv 接受所有的参数 保存在数组中

yield

进阶

  • 生成器函数， 可以理解为暂停，程序会暂停在yield的地方， 等待下一次调用 next() 时， 程序又会执行一次， 然后继续执行

next()

可以通过打断点来进行理解， 让程序一步一步执行， 查看程序到底执行到了那里， 暂停到了那里
